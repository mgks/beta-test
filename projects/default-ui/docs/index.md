# Default UI + Showcase

Welcome to the **Default UI + Showcase** test project. This project serves as a comprehensive suite to verify all the core rendering and layout features of `docmd`.

## Features Tested in this Project

- **Layout Structure:** Header, Sidebar navigation, Breadcrumbs, and a complete Footer.
- **Top Menubar:** Navigation links at the top of the viewport.
- **Plugins:** Mermaid flowcharting and LaTeX math rendering.
- **Typography and Elements:** Headings, blockquotes, lists, tables, and standard HTML structures.
- **Callout Containers:** Both standard callouts and leaking/malformed callouts to verify parser security and robustness.
